Введение в Git

Git — это система контроля версий, позволяющая отслеживать изменения в коде и работать над проектами в команде. Она была создана Линусом Торвальдсом для управления разработкой ядра Linux.

Установка Git

# macOS
brew install git

# Ubuntu / Debian
sudo apt install git

# Windows
Скачать установщик с https://git-scm.com

Настройка Git

git config --global user.name "Ваше Имя"
git config --global user.email "email@example.com"

# Проверка настроек
git config --list

Основной рабочий процесс

  1. Создать репозиторий: git init
  2. Добавить файлы: git add .
  3. Сделать коммит: git commit -m "Первый коммит"
  4. Посмотреть историю: git log

Ветки и слияния

# Создать новую ветку
git branch feature

# Переключиться на неё
git checkout feature

# Объединить ветку с основной
git merge feature

Работа с удалёнными репозиториями

# Добавить удалённый репозиторий
git remote add origin https://github.com/user/repo.git

# Отправить изменения
git push -u origin main

# Получить изменения
git pull origin main

Полезные команды

# Отмена последнего коммита, сохранив изменения
git reset --soft HEAD~1

# Просмотр различий
git diff

# Удаление ветки
git branch -d feature
